# CubePixel

## Getting Started 
1) Install Python 3.7 or newer.
2) Install the dependencies with `pip install -r requirements.txt`.
3) Build the cython file with `python setup.py build_ext --inplace`.
4) Run [/CubePixel/main.py](/CubePixel/main.py).
